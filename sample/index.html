Testing ckeditor5 with mathpreview plugin

Mathpreview plugin for ckeditor5 allows you to preview how a latex math formula will look like, when rendered. It supports MathJax and Katex engines. The plugin works like this:
		
     
			
  •  The toolbar button f(x) operates as a ON/OFF switch. For the plugin to work the button must be enabled.
    • 
			
  •  Enter a formula in latex, enclosed in \[ \] or \( \). For example \[x^2+5x-12=0\]
    • 
			
  •  Click on the typed formula. A popup should display it rendered.
